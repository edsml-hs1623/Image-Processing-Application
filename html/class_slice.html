static

Extracts and saves a slice from the given volume along the XZ plane at a specified Y index.

Parameters
volumeThe 3D volume from which to extract the slice.
yThe index along the Y-axis at which to extract the slice. Must be within the bounds of the volume's height.
outputPathThe filesystem path where the resulting slice image will be saved as a PNG file.
Note
The function asserts that the given Y index is within the volume's bounds.
